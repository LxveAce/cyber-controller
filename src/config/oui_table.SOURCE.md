# `oui_table.tsv.gz` — provenance

Compact `PREFIX<TAB>Organization` mapping (gzipped, sorted) of the 24-bit IEEE **MA-L** OUI registry,
used by `src/core/oui.py` to label scanned devices with their manufacturer.

- **Source:** IEEE Registration Authority — <https://standards-oui.ieee.org/oui/oui.csv> (free, public).
- **Retrieved:** 2026-07-13.
- **Generated by:** `scripts/gen_oui_table.py oui.csv` (drops `Private`/withheld blocks; keeps
  `Assignment → Organization Name` only — no addresses).
- **Contents:** the OUI→organization *assignments* only. This is factual registry data (as bundled by
  Wireshark, nmap, aircrack-ng, …); redistributed with attribution to IEEE.
- **Refresh:** re-download `oui.csv` and re-run the generator, or load a fresh copy at runtime via
  `oui.load_ieee_csv(path)` / `oui.load_manuf(path)` — no rebuild needed.
